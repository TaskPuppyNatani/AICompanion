NOTIFICATION LOOP PREVENTION
=============================

RISK: If companion sends notifications, listener captures them, creating loop.

MITIGATION (Phase 2 implementation):

def _on_notification_changed(self, sender, args):
    app_name = self._extract_app_name(args.UserNotification)
    
    # Filter: Skip self-notifications
    if self._is_self_notification(app_name):
        return
    
    # Filter: Skip spam/system notifications
    if self._is_noise(app_name):
        return
    
    # Safe to process
    self._post_to_rivet(args.UserNotification)

def _is_self_notification(self, app_name):
    # Companion app will have specific app ID when packaged
    companion_ids = [
        'AICompanion',
        'companion',
        sys.argv[0].lower(),
    ]
    return any(c in app_name.lower() for c in companion_ids)

TROUBLESHOOTING
===============

ERROR: "Failed to import winsdk: No module named 'winsdk'"
FIX: Run: pip install winsdk

ERROR: "Failed to get listener: [WinError -2147418105] ..."
FIX: This usually means:
  - You're on Windows < 10 (need Windows 10+)
  - Notifications are disabled globally
  Try: Settings > Privacy & security > Notifications > Make sure it's ON

ERROR: "No output even with notifications enabled"
FIX: 
  - Notifications must appear as VISIBLE TOASTS (bottom-right corner)
  - If app has notifications muted, they won't show as toasts
  - Check: Settings > Notifications & actions > App notifications
  - Make sure Discord/Telegram/Outlook have notifications ON

ERROR: "Script exits immediately with no error"
FIX: Check your Python version
  python --version
  (Need 3.8+; if older, upgrade Python)

PERMISSION ISSUE: Script runs but gets no notifications
FIX:
  - Settings > Privacy & security > App permissions > Notifications
  - Make sure Discord, Telegram, and Outlook are toggled ON
  - Each app needs explicit permission granted

==========================
